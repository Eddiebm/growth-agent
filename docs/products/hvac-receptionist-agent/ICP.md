# ICP — HVAC AI Receptionist

## Target

- **Industry:** HVAC, plumbing, home services, contractors
- **Size:** 3–100 employees
- **Geo:** US
- **Titles:** Owner, General Manager, Operations Manager

## Qualifiers

- Takes inbound service calls daily
- Misses after-hours calls
- Uses dispatch software or spreadsheets

## Disqualifiers

- Commercial-only industrial (different sales motion)
- No phone-based booking

## Pain points

1. Missed calls = lost jobs
2. Receptionist overtime costs
3. Inconsistent call handling quality

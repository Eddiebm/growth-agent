# ICP — Boswell Cloud

## Target

- **Industry:** Software companies, dev agencies, SaaS consultancies
- **Size:** 5–500 employees
- **Geo:** US, UK, Canada
- **Titles:** CTO, VP Engineering, Head of Engineering, Founder

## Qualifiers (must have ≥2)

- Maintains code on GitHub or GitLab
- Onboards engineers or contractors regularly
- Has documentation debt or handoff friction

## Disqualifiers

- No engineering team
- Pure non-technical SMB
- Competitor building similar audit tools

## Pain points

1. Codebases are opaque to new hires and clients
2. Handoff docs are always out of date
3. Audits are manual and expensive
